//Crea una classe que llisti alfabèticament el contingut d'un directori rebut per paràmetre.//

Haurem de compilar i fer un build primer de tot per poder-ho executar.

Per executar fes un run i et demanara la ruta del directori a llistar tambe ho pots fer amb commands via terminal.

En aquest exercici podem comprobar com es llista un arbre de directoris en ordre alfabetic i mostrant tambe la seva data de modificacio tambe indica si es un directori amb la lletra D o un fixter amb la lletra F.
